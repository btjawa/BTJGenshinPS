## Building

使用NPM构建

**Requirements:**

 - [Node.js](https://registry.npmmirror.com/binary.html?path=node/v18.16.1/) >= 18.16.1
 > npm >= 9.5.1
 - [Git](https://git-scm.com/downloads)

```shell
git clone https://github.com/btjawa/BTJGenshinPS.git
cd BTJGenshinPS
npm install # 安装依赖
```

**Scripts**

```shell
npm run start # 预览
npm run package # 打包
```
